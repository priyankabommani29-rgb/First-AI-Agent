# 🤖 My First AI Agent using Groq API

Welcome to my **first AI Agent project!** 🎉

This project marks the beginning of my journey into AI Agent development. It is an interactive command-line AI assistant built with **Node.js** and the **Groq API**, demonstrating how Large Language Models (LLMs) can understand user queries, call custom functions, and generate intelligent responses.

While this is my first AI Agent, it laid the foundation for learning concepts such as tool/function calling, API integration, and conversational AI.

---

## 🚀 Features

* 🤖 AI-powered conversational assistant
* 🛠️ Function/Tool Calling support
* 💬 Interactive command-line interface
* ⚡ Powered by Groq LLMs
* 📚 Beginner-friendly and easy to extend

---

## 🛠️ Tech Stack

* Node.js
* JavaScript (ES Modules)
* Groq API
* OpenAI SDK (configured for Groq)
* readline-sync

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-agent.git
cd ai-agent
```

Install dependencies:

```bash
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key_here
```

> Never commit your API keys to GitHub.

---

## ▶️ Run the Project

```bash
node index.js
```
